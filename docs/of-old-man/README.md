# Of Old Man

A poetry collection featuring diverse poetic forms and styles, generated twice daily.

## About

"Of Old Man" is an ongoing poetry series that explores the full spectrum of poetic expression. Twice each day, a new poem is generated in a randomly selected poetry form, showcasing the rich diversity of poetic traditions from around the world.

## Poetry Forms

The collection draws from 91 different poetry types, including:

- Classical forms: Sonnet, Haiku, Villanelle, Sestina, Ballad
- Modern styles: Free verse, Slam poetry, Spoken word, Performance poem
- Specialized forms: Ghazal, Pantoum, Terza rima, Rondeau, Sijo
- And many more...

## Generation Schedule

- **Frequency**: Twice daily (10:00 UTC and 22:00 UTC)
- **Format**: Individual poems in their respective traditional forms
- **Style**: Authentic to each poetry type's conventions
- **Length**: Varies by form (no artificial constraints)

## Features

- **No Word Count Limits**: Each poem follows the natural length and structure of its form
- **Authentic Forms**: Poems honor the traditional techniques, structures, and styles of each poetry type
- **Continuous Series**: This collection never ends - new poems are added twice daily
- **Timestamped**: Each poem includes its publication date and time
- **Diverse Styles**: Randomly selected poetry types ensure variety and exploration

## Technical Details

- **Generated by**: Google Gemini AI (gemini-2.5-flash)
- **Storage**: GitHub Gist
- **Automation**: GitHub Actions (twice daily workflow)
- **Configuration**: `scripts/of-old-man/poetry_types.json`
